# 题目描述

下面的图形是著名的杨辉三角形：

![yh](file://yh.jpg)

如果我们按从上到下、从左到右的顺序把所有数排成一列，可以得到如下数列：

$1, 1, 1, 1, 2, 1, 1, 3, 3, 1, 1, 4, 6, 4, 1, \cdots$

给定一个正整数 $N$，请你输出数列中第一次出现 $N$ 是在第几个数？

# 输入格式

输入一个整数 $N$。

# 输出格式

输出一个整数代表答案。

# 输入输出样例

```input1
6
```

```output1
13
```

# 说明/提示

【数据规模】

对于 $20\%$ 的评测用例，$1 \leq N \leq 10$；

对于所有评测用例，$1 \leq N \leq {10}^9$。
